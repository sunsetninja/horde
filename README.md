# horde
RCV logs formatter

## Installation

```sh
  make setup
```

## Usage

To see a list of commands that GenDiff offers, you can run:

```sh
  horde --help
```

### Usage signature

```sh
  horde [options] <filepath>
```

### Usage examples

```sh
  horde --day 2022-09-12 Hercules_RWC_Logs.log 
```
